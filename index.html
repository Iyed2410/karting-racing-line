Karting Racing Line

Visualize, optimize and simulate the fastest smooth racing line for kart tracks — browser-based, lightweight, and interactive.

Open App → Read Docs

What it does

Draw a centerline, tune kart physics, and generate an optimized racing line that balances speed and smoothness while keeping the kart on track.

Key Features

  • Interactive track editor — draw, edit, undo/redo
  • Simulated-annealing optimizer with dynamics-aware scoring
  • Speed heatmap, apex detection, and playback simulation
  • Optional WebGL renderer for smoother visuals on large tracks
  • High-DPI canvas support and responsive UI

How to Use

  1. Open the app and draw the track on the canvas.
  2. Adjust grip / acceleration / braking to match your kart.
  3. Click Generate Racing Line to optimize.
  4. Toggle Speed Heatmap or Apexes to inspect the line.
  5. Use playback to visualize the lap and export the track as JSON.

Design Goals

Produce a realistic, drivable racing line that minimizes lap time while remaining smooth and physically plausible. The optimizer considers lateral grip, acceleration/braking limits, and penalizes excessive curvature.

Shortcuts

  • Space — Play/pause simulation
  • +/- — Zoom
  • F — Fit view
